# 📱 Projeto Android

Projeto desenvolvido durante o **Módulo 02 do curso de HTML5 e CSS3** do Curso em video

O objetivo do projeto foi criar uma página web responsiva utilizando os fundamentos de **HTML5 e CSS3**, aplicando conceitos de estruturação semântica, organização de código e adaptação para diferentes tamanhos de tela.

- 🔗 Acesse o projeto: [Link do projeto](https://devgustavojeferson.github.io/Projeto_Android/)

## 🎯 Objetivo do Projeto

Criar uma página informativa sobre o surgimento do sistema operacional Android, praticando:

- Estruturação de páginas com HTML5
- Estilização avançada com CSS3
- Organização de arquivos
- Responsividade
- Boas práticas de desenvolvimento

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Visual Studio Code (IDE)

---

## 📚 Principais Aprendizados

Durante o desenvolvimento aprendi na prática:

### 📝 Planejamento

- Criar um rascunho antes de iniciar o código
- Definir paleta de cores, fontes e imagens previamente
- Organizar melhor o processo de desenvolvimento

---

### 🏗️ HTML Semântico

Utilização de elementos semânticos:

- `header`
- `nav`
- `main`
- `article`
- `aside`
- `footer`

Melhorando a estrutura e a leitura do documento.

---

### 🎨 CSS

Aplicação de:

- Variáveis CSS com `:root`
- Seletores avançados
- Pseudo-classes
- Pseudo-elementos
- Sombras
- Bordas
- Formatação de textos
- Organização por seções

---

### 📱 Responsividade

Adaptação do site para diferentes dispositivos utilizando:

- `width: 100%`
- `max-width`
- `picture`

Garantindo uma melhor experiência em telas menores.
